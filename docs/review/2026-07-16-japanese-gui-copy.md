# 日本人ユーザー向けGUI文言見直し — 統合記録

## 対象

VS Code拡張で利用者に表示される設定、ワークフロー、テストスイート、コマンド、入力、確認、通知、エラーの文言を自然な日本語へ統一する。

コマンドID、設定キー、JSONキー、ファイル名、CLI引数、ARIA属性、処理フロー、状態遷移は変更対象外とする。

## 実装境界

- 設定項目名、説明、状態表示、操作ラベル
- 詳細／簡易ワークフローの工程名と補助操作
- テストスイートの見出し、集計、列名、実行状態
- コマンドパレット、設定説明、入力欄、ファイル選択、確認ダイアログ、通知、エラー
- 既存テストの期待文言と新しいGUI文言契約

## 自動適用の後処理

一時的に使用したパッチ展開ファイル、失敗ログ、変換スクリプト、テストランナー差し替え、エクスポート用ワークフローは最終製品差分から除去した。

## 統合ゲート

この記録を含む正確なPRヘッドについて、次のGitHub Actionsジョブがすべて成功してから `main` に統合する。

- Source integrity
- Python tests
- VS Code unit tests
- VS Code Extension Host activation
- VC6 fixture smoke
- Package contract
